Detailed Description

template<class C, template< typename > class V>
struct util::type_map< C, V >

Given a container of types C<T1,T2,T3,...> and template template type V<T>, this function will apply the types in C to V<T>. This produces C<V<T1>, V<T2>, V<T3>, ...>.
